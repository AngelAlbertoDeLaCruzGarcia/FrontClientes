# Aplicacion de React

### Pasos para ejecutar el proyecto
 

* Clonar el proyecto
```
git clone https://github.com/AngelAlbertoDeLaCruzGarcia/FrontClientes.git
```
* Instalar los paquetes
```
npm install
```
* Ejecutar el proyecto
```
npm run dev
```
